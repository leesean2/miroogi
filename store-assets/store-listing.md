# Play 스토어 등록 정보 (초안)

그대로 복사해서 Play Console에 붙여넣으면 됩니다. 필요하면 다듬어 쓰세요.

## 앱 이름 (최대 30자)
```
미루기 - 미루기 패턴 트래커
```

## 짧은 설명 (최대 80자)
```
완료가 아니라 '회피'를 기록해요. 점수도 비교도 없이, 미루는 나를 판단 없이 관찰하는 앱.
```

## 자세한 설명 (최대 4000자)
```
미루기는 '할 일을 다 했는지'가 아니라 '무엇을 미뤘는지'를 기록하는 앱입니다.
잘잘못을 따지지 않고, 그저 나의 미루기 패턴을 관찰하는 데 집중합니다.

점수도, 연속 기록(스트릭)도, 남과의 비교도 없습니다.
다그치는 대신, 먼저 나를 알아차리는 것부터 시작해요.

■ 기록
미룬 일을 판단 없이 그대로 적어요. 3초면 충분합니다.
카테고리(업무·개인·학업 등)와 미룬 이유는 적어도, 안 적어도 괜찮아요.
필요하면 나만의 카테고리와 이유도 직접 추가할 수 있습니다.

■ 패턴
기록이 쌓이면 최근 6주의 요일별 히트맵으로 정리돼요.
'어떤 요일에, 어떤 이유로 자주 미루는지'가 눈에 보이기 시작합니다.
"월요일에 유독 무언가를 미루는 경향이 보여요" 같은 작은 관찰을 건네줍니다.

■ 실험
패턴에서 발견한 것을 일주일짜리 작은 실험으로 가볍게 시도해봐요.
성공/실패를 따지지 않고, 그냥 '해봤는지'만 도트로 체크하면 됩니다.

■ 오롯이 내 기기 안에서
모든 기록은 당신의 기기 안에만 저장됩니다.
회원가입도, 로그인도, 서버 전송도 없습니다.
광고와 추적도 없습니다. 오프라인에서도 잘 동작해요.

미루는 습관을 미워하지 말고, 조용히 관찰하는 것부터 시작해보세요.
```

## 분류/설정 제안
- 카테고리: 라이프스타일 (또는 자기계발/생산성)
- 태그: 습관, 자기관리, 생산성, 다이어리
- 광고 포함: 아니요
- 인앱 구매: 아니요

## 데이터 보안(Data Safety) 양식 답변 요약
